# UI設計

## 画面構成

```
[ホーム] → [路線詳細]
```

---

## 画面1: ホーム（路線選択）

### 概要
JR東日本の路線一覧をカード形式で表示。

### 要素
- ヘッダー: アプリ名
- 路線カード一覧
  - 路線名（例: 山手線）
  - 現在の状況アイコン（🟢正常 / 🟡遅延 / 🔴運転見合わせ）
  - 簡易リスクスコア
- フッター: コンテスト表記

### アクション
- カードタップ → 画面2へ遷移

---

## 画面2: 路線詳細（リスク分析）

### 概要
選んだ路線の詳細リスク情報を表示。

### 要素
- 戻るボタン
- 路線名
- リアルタイム状況（テキスト）
- リスクスコア（ドーナツチャート）
- 時間帯別傾向グラフ（棒グラフ）
- アドバイステキスト
- 「別の路線を見る」ボタン
