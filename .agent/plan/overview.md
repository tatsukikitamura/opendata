# 遅延リスク予測アプリ

## 目的
**「今日、この路線は遅延しそうか？」をひと目で分かるようにする。**

## スコープ

### ✅ やること
- JR東日本の主要路線の遅延リスク表示
- リアルタイム運行情報の取得・表示
- 過去の遅延傾向の可視化（時間帯別など）

### ❌ やらないこと
- 経路探索（A駅→B駅の行き方）
- 他社線（東京メトロ、私鉄など）
- 運賃計算

## 技術スタック
| レイヤー | 技術 |
|---|---|
| フロントエンド | Vite + Vanilla JS + Tailwind CSS v4 |
| バックエンド | FastAPI (Python) |
| データベース | SQLite |
| 外部API | ODPT API |
